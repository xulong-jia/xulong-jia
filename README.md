## Featured Project

### [YOLOv8 Vehicle and Pedestrian Detection System](https://github.com/xulong-jia/yolov8-vehicle-pedestrian-detection)

Final release: `v1.8.6-final-submission-report`

An engineering-focused computer vision system for vehicle and pedestrian detection. It is designed as a complete project workflow rather than a single YOLO script: data checks, model experiments, image/video inference, tracking, analytics, API service, local UI, Docker deployment, documentation, and acceptance checks.

Key capabilities:

- YOLOv8 image and video inference for vehicle and pedestrian classes.
- ByteTrack multi-object tracking with `tracks.csv` output.
- Line crossing, ROI counting, and lightweight event analytics.
- VideoAnalysisCenter for structured result management and artifact summaries.
- FastAPI service with async video jobs and registered artifact downloads.
- SQLite-backed job metadata for video analysis tasks.
- Streamlit demo and minimal React frontend for local operation.
- Docker deployment with mounted model weights rather than committed weights.
- Bad Case review workflow and reviewed GT evaluation sample pack.
- Non-technical macOS/Windows launcher scripts for prepared local environments.

Scope notes:

- DeepSORT is documented as optional future work.
- The React frontend is minimal and optional, not a production dashboard.
- The reviewed GT evaluation is a small sample pack, not a large-scale benchmark.
